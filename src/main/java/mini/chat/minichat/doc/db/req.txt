API:
1. Đăng ký user (đăng kí bằng sđt, fullname, avartar, password, giới tính) done
2. Đăng nhập (Số điện thoại, mật khẩu) done
3. Tìm kiếm user theo số điện thoại. done
4. Gửi lời mời kết bạn (status bằng 1 là gửi lời mời r nhưng chưa chap nhận, bằng 2 là đang là bạn bè) done
5. Chấp nhận kết bạn (chuyển status bằng 2) done
6. Lấy danh sách chat_rooms của mình(user đang đăng nhập) có tham gia
7. Lấy danh sách bạn bè của mình (status = 2) done
